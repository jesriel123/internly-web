# 🔐 How to Reset Your Password

## ⚡ Quick Steps

### 1. Click the Reset Link
Click the full URL from your email:
```
https://internly-web.vercel.app/reset-password#access_token=...
```

### 2. Enter New Password
- Type your new password (minimum 6 characters)
- Confirm by typing it again
- Click "Update Password"

### 3. Login
- You'll be redirected to login
- Use your email and new password

**Done!** 🎉

## ❌ If It Doesn't Work

### Problem: "Link expired"
**Solution**: Request a new link
1. Go to https://internly-web.vercel.app/login
2. Click "Forgot Password"
3. Enter your email
4. Check email for new link

### Problem: "Invalid link"
**Solution**: Copy the FULL URL
- Make sure you copy everything including `#access_token=...`
- Paste in browser address bar
- Press Enter

### Problem: Page shows error
**Solution**: Try these:
1. Use different browser (Chrome, Firefox, Edge)
2. Try incognito/private mode
3. Clear browser cache
4. Disable VPN if using one
